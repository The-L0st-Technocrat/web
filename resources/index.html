Cybersecurity Fundamentals Live Training Resources

Instructor: Omar Santos
Twitter: @santosomar

The following are additional resources that enhance the learning during the live training class. In this course you will learn introductory cyber security concepts, as well as the different focus areas in cyber security. You will explore different concepts of incident response, digital forensics, security operations, and security management. This course will also go over cyber security in the Internet of Things (IoT), secure development management, and how cyber security is affecting software development (including DevOps and cloud computing).
